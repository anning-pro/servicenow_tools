# Privacy Policy / プライバシーポリシー

Last updated: 2026-02-28

This policy applies to the Chrome extension **Excel to ServiceNow**.
本ポリシーは Chrome 拡張機能 **Excel to ServiceNow** に適用されます。

## 1. Data We Collect / 収集する情報

- Chrome account identifier information via `chrome.identity` / `chrome.identity.email` (email address or account id)
  - `chrome.identity` / `chrome.identity.email` による Chrome アカウント識別情報（メールアドレスまたはアカウントID）
- Extension settings entered by the user (for example: instance URL, username, mapping settings, language)
  - ユーザーが入力した拡張機能設定（例: インスタンスURL、ユーザー名、マッピング設定、言語）
- Usage count data (used to manage feature usage limits)
  - 利用回数データ（機能利用回数の管理に使用）

## 2. How We Use Data / 利用目的

We use data only to provide core extension functions:
データは、拡張機能の主要機能提供のためにのみ使用します。

- identify the current user and apply per-user usage limits
  - ユーザー識別とユーザー単位の利用回数管理
- save and restore extension settings
  - 拡張機能設定の保存と復元
- validate ServiceNow instance context and execute requested operations
  - ServiceNow インスタンス整合性確認とユーザー操作の実行
- display operation results (success/failure notifications)
  - 実行結果（成功/失敗通知）の表示

## 3. Data Sharing / 第三者提供

We do not sell personal data.
個人データを販売することはありません。

Data is shared only when necessary for functionality:
機能提供に必要な範囲でのみ、以下と通信します。

- ServiceNow instance (`https://*.service-now.com/*`) to create/update table and column definitions requested by the user
  - ユーザー操作に基づくテーブル/カラム定義の作成・更新のため、ServiceNow インスタンス（`https://*.service-now.com/*`）
- ExtensionPay/ExtPay (`https://extensionpay.com/*`) to verify payment/subscription status
  - 課金・サブスクリプション状態確認のため、ExtensionPay/ExtPay（`https://extensionpay.com/*`）

## 4. Storage and Retention / 保存場所と保持

- Settings and usage data are stored in Chrome extension storage (`chrome.storage.local` / `chrome.storage.sync`).
  - 設定情報および利用回数情報は Chrome 拡張機能ストレージ（`chrome.storage.local` / `chrome.storage.sync`）に保存されます。
- Data is retained until the user changes settings, clears browser extension data, or uninstalls the extension.
  - データは、ユーザーが設定変更・拡張機能データ削除・拡張機能アンインストールを行うまで保持されます。

## 5. User Controls / ユーザーによる管理

Users can:
ユーザーは以下を実行できます。

- edit or delete settings from the extension UI
  - 拡張機能UI上で設定の変更・削除
- clear extension storage via Chrome settings
  - Chrome 設定から拡張機能データの削除
- uninstall the extension at any time
  - いつでも拡張機能をアンインストール

## 6. Security / セキュリティ

We take reasonable measures to protect stored settings and transmitted data.
保存データおよび通信データの保護のため、合理的な安全対策を講じます。

## 7. Changes to This Policy / 改定

We may update this policy as needed. The latest version will be published at the same URL.
本ポリシーは必要に応じて改定されます。最新版は同一URLで公開します。
